# Gestionnaire des Étudiants en C

## Description

Le **Gestionnaire des Étudiants** est une application développée en langage C permettant de gérer une liste d'étudiants à travers une interface en ligne de commande.

L'application permet d'ajouter, d'afficher, de rechercher, de modifier et de supprimer des étudiants. Elle permet également d'afficher l'étudiant ayant la meilleure moyenne et de calculer la moyenne générale de la classe.

Les données sont stockées dans un tableau de structures (`struct`) pendant l'exécution du programme.

Ce projet a été réalisé dans le cadre de mon apprentissage du langage C afin de mettre en pratique les notions de structures, de tableaux, de fonctions, de pointeurs et de manipulation des chaînes de caractères.

---

## Fonctionnalités

* Ajouter un étudiant
* Afficher tous les étudiants
* Rechercher un étudiant par son nom
* Modifier un étudiant
* Supprimer un étudiant
* Afficher l'étudiant ayant la meilleure moyenne
* Calculer la moyenne générale
* Quitter le programme
